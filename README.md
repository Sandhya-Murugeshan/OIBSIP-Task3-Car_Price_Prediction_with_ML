# OIBSIP-Task3-Car_Price_Prediction_with_ML

# 🚗 Car Price Prediction using Machine Learning

## 📌 Project Overview
This project focuses on predicting the price of a car based on its features such as fuel type, engine size, and horsepower.  
It is a simple machine learning regression problem where we estimate continuous values (price).

---

## 🎯 Objective
The main goal is to:
- Understand how machine learning can predict prices
- Train a regression model on car data
- Use the trained model to predict prices for new cars

---

## 📊 Dataset Features
Some important features used in this project:
- **fueltype** → Gas or Diesel  
- **enginesize** → Engine capacity  
- **horsepower** → Engine power  

Target variable:
- **price** → Car price (what we predict)

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- Scikit-learn  

---

## 🧠 Machine Learning Model
We used:
- **Random Forest Regressor**

Why?
- Works well with both small and large datasets  
- Handles non-linear relationships  
- Gives good accuracy without complex tuning  

---

## 🔄 Project Workflow
1. Load the dataset  
2. Perform basic preprocessing  
3. Encode categorical data  
4. Split data into training and testing sets  
5. Train the model  
6. Evaluate performance  
7. Predict price for new cars  

---

## 📈 Model Evaluation
The model is evaluated using:
- **RMSE (Root Mean Squared Error)**
- **R² Score**

These metrics help us understand how accurate the predictions are.

---

## 🚀 Sample Prediction
The model can predict prices for different cars like:

Example:
- Fuel Type: Gas  
- Engine Size: 130  
- Horsepower: 111  

➡️ The model outputs the estimated car price.

## 💡 Key Learnings

- Practiced real-world prediction problem

## 📌 Conclusion
This project demonstrates how machine learning can be used to predict car prices based on key features.  
It is a beginner-friendly project to understand regression and model building.
